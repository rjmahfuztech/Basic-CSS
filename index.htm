Basic CSS

CSS:
Cascading Style Sheets

Declaration of CSS :

There are three ways to declare CSS as follows :

  1. Inline
  2. Internal
  3. External

A. Inline CSS:
To apply unique style of a single HTML tag an Inline CSS is very effective. We can use Inline CSS by Style attribute in HTML tag.
Syntax of Inline CSS:
In case of single value of style :
<tag style="property:value"> Element of HTML tag </tag >
 In case of Multiple values of style :
< tag style="property:value;property:value;property:value"> Element of HTML tag </tag>
**Designed by Inline CSS

B. Internal CSS:
To apply same style of HTML tag in a HTML page. Internal CSS is declare in head section of HTML.
Syntax of Internal CSS:
In case of single value of style :
< style >
selector { property:value }
</style >
In case of Multiple values of style:
< style > selector { property:value; property:value; property:value; }
< /style >
**Designed by Internal CSS

C. External CSS:
To apply style of many HTML pages an External CSS is very popular and effective for indentification. External CSS file is linked up head section of HTML.
Syntax of External CSS:
In head section of HTML:
< link rel="stylesheet" href="location of file is inserted here" />
External style file is designed by:
  In case of single value: selector {property:value}
  In case of multiple values: selector {property:value;property:value;property:value}
**Designed by External CSS
